
# Welcome to JekyllPad's Free Online Markdown Editor

This powerful WYSIWYG editor supports GitHub-Flavored Markdown (GFM). Here's a showcase of its features:

## Text Formatting

You can make text **bold**, _italic_, ~strikethrough~, or `inline code`.

## Lists

### Unordered List

*   Item 1
    
*   Item 2
    
    *   Subitem 2.1
        
    *   Subitem 2.2
        

### Ordered List

1.  First item
    
2.  Second item
    
3.  Third item
    

## Task Lists

*   Completed task
    
*   Incomplete task
    

## Headings

# H1 Heading

## H2 Heading

### H3 Heading

#### H4 Heading

##### H5 Heading

###### H6 Heading

## Blockquotes

> This is a blockquote. It can span multiple lines and can contain other Markdown elements.
